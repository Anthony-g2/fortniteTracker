# fortniteTracker
Fortnite Tier Tracker


Use this javascript file to calculate how many tiers and battle-stars you'll recieve while leveling up. 

Examples:

```
levelTiers(1,0,100)
>> 30 tiers 8 stars.
```
In the above sample:
```
Starting Level = 1,
Starting Battle Stars = 0,
Level Goal = 100
```
